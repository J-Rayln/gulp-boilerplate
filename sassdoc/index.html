    font-size

    @mixin font-size($size, $base: 16) { ... }

    Description

    Outputs px based font sizes to rems with px fallback

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    Font size in pixels. Do not include the 'px' unit

    		Number— none
    $base

    Base font size. Assumes 1rem = 16px.

    		Number16

    Example

    outputs a font size of 1.125rem with 18px fallback.

    @include font-size(18);

    Output

    font-size: 18px;
    font-size: 1.125rem;

    breakpoints

    font-sizes

    $font-sizes: (
    "body": (
        "min": rem(14),
        "max": 1rem 
    ),
    "h1": (
        "min": 1.25rem,
        "max": 2rem
    ),
    "h2": (
        "min": .875rem,
        "max": 2rem
    ),
    "h3": (
        "min": 1rem,
        "max": 1rem
    ),
);

    Description

    Min/Max font sizes used for fluid font sizes. Values in this map are used to build clamp() values for CSS custom properties stored in the :root element.

    • Font sizes MUST be in rems for clamp function to work.
    • Ff there is no min/max, set the 'min' and 'max' to the SAME value.
    • Comment out values you don't need or add more using the same format.
